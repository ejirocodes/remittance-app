# remittance-app


## Set Node Version
This project requires Nodejs version 14, so please run the following command!
```
nvm use
```
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
